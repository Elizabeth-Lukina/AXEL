<p align="center"> Axiom is a smart personal assistant Telegram bot that helps you stay organized.
It manages your to-do list and sends you a personalized morning digest at your chosen time — including weather, currency rates, inspirational quotes, and more.
Designed to think ahead, speak your language, and evolve with you.</p>

## About AXEL
- 🌤 Daily weather, 💱 exchange rates, 🧠 thought of the day + your 📋 tasks
- 🗓 **Planner mode** — manage tasks without leaving the task menu until you choose to return
- ⏰ **Daily report management** — change time, contents, or disable the digest anytime
- 🎮 **Tic-Tac-Toe** game with the bot — quick entertainment when you need a break
- ✍️ Natural language task input (e.g., “Buy bread tomorrow at 10:30”)
- 🧵 Voice message support, AI-powered replies, and feedback system
- 📊 Personal statistics and analytics
- 🎯 Future goals: voice interaction, smart home integration, full J.A.R.V.I.S prototype


## Tech Stack
- 🐍 Python 3.10+  
- 🤖 Telebot (pyTelegramBotAPI)  
- 🗃 SQlite3  
- 📅 APScheduler  
- 📈 Matplotlib / Pandas  
- 🔐 python-dotenv  
- 🌦 Weather API: OpenWeatherMap  
- 💱 Currency: exchangerate.host  
- 🧠 NLP: dateparser / nltk  

## What's Coming Next
- 🔊 Voice input & speech recognition in the task menu
- 📝 Notes feature to track ideas and reminders
- 🎲 Fun mini-games for breaks and engagement
- 💰 Finance management tools (in development)
- 🔄 Google Calendar sync
